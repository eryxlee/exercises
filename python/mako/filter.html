Mako模板语言示例 - 过滤器

1. 默认的escape

1. The value is: ${curl}
这种使用方式相应的python代码是 escape(curl)

2. u过滤，主要用于构成URL

2. The value is: ${curl|u,n}
这种使用方式相应的python代码是 filters.url_escape(curl)

3. h过滤，主要用于构成HTML，基本等同默认的escape

3. The value is: ${curl|h,n}
这种使用方式相应的python代码是 filters.html_escape(curl)

4. x过滤，主要用于构成XML

4. The value is: ${curl|x,n}
这种使用方式相应的python代码是 filters.xml_escape(curl)

5. trim过滤，去掉前后空格

5. The value is: ${curl|trim,n}
这种使用方式相应的python代码是 filters.trim(curl)

6. entity过滤，HTML entity转换

6. The value is: ${curl|entity,n}
这种使用方式相应的python代码是 filters.html_entities_escape(curl)

7. n过滤，去掉所有默认的过滤

7. The value is: ${curl|n}
这种使用方式相应的python代码是 curl
